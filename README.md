# paintmoji
An example of a realtime collaborative app based on ShareDB and Operational Transformation.

## Setup

1. Install [Node.js](https://nodejs.org/en/download).
2. Run `npm install`

## Usage

1. Run `npm start`.
2. Open [http://localhost:3000](http://localhost:3000) on your browser.
